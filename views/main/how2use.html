어서오세요!

이 웹사이트는 pc, 모바일, 태블릿 pc 화면에 최적화되어있으며, 사파리, 크롬 브라우저 등에서 정상 동작합니다.

사용법

1. 점호 시작 전 로그인하여 '점호하러 가기' 를 누릅니다.

2. 외박자 명단을 확인 후 '정상' 에 체크합니다. (정상은 정상 외박 처리를 뜻합니다.)

3. 각 호실을 점검하며 사생 상태를 체크합니다.

3-1. 이때, '정상'에서 '재실' 등으로 상태 변경 시 체크박스를 해제하셔야 합니다.

4. 점호 보고가 끝난 후, '저장 내역 삭제'를 눌러주세요.

점호 도중에 정보가 유실됐어요 ㅠㅠ

침착하게 목록 맨 밑의 '불러오기' 를 눌러주세요. 유실된 체크박스가 다시 돌아옵니다.

왜 '저장 내역 삭제'를 눌러야 하나요?

점호 도중 사생의 상태 정보(외박, 점호불응, 재실 등)가 사라지는 것을 막기 위해, 매 상태 체크 시 모든 정보가 저장됩니다.

따라서 점호 도중에 페이지 새로고침을 한다던가 인터넷이 유실되어도, '불러오기'를 누르면 모든 정보가 돌아옵니다.

이때, 저번 주 점호의 저장 내역을 삭제하지 않았다면, 저번 주 점호의 정보까지 돌아옵니다.

때문에 꼭 점호 보고가 끝난 후, 혹은 외박자 기록 전에 '저장 내역 삭제'를 눌러주세요.
