# 🏢 Roll call website for Hongik Dormitory
- 출석부와 유사한 인원 체크 웹 사이트  
- link: https://hongik-dorm.herokuapp.com/  
- 회원 가입 후 사용 가능
## 💍 Features
- 층장 회원가입과 정보 수정
- 담당 층 사생 명단 추가, 삭제, 수정
- 담당 층 인원체크(재실, 점호 불응, 외박, 무단 외박)
